# Web App Features

## Platform & Workspace
- Multi-organization support with isolated workspaces
- Program-based workspaces with event templates
- Role-based access control with granular permissions
- Audit logging for critical actions

## Program Setup & Management
- Program creation with event types and metadata
- Program details (deadline, visibility, branding)
- Public program landing pages with sections
- Category management (nested categories)
- Submission process configuration
- Form builder with multi-step sections

## Submissions & Entries
- Submission intake (internal + public)
- Submission status lifecycle (Pending, Under Review, Shortlisted, Accepted, Rejected)
- Bulk actions (status updates, judge assignment, delete)
- Submission detail view with judge assignments
- File attachments support

## Judging & Review
- Judge roster management (add/invite)
- Judge assignments to submissions
- Judging criteria configuration
- Multi-round judging workflows
- Shortlist handling and visibility controls
- Judging progress and completion tracking

## Schedule & Rounds
- Program schedule with start/end dates
- Workflow-based round editor (tile + workflow views)
- Round configuration (logic, timing, eligibility)
- Fixed start/end time enforcement
- Round ordering and versioning

## Overview & Analytics
- Dashboard overview with key stats
- Submission trends charts
- Category distribution breakdowns
- Navigation from overview cards to details

## Teams & Roles
- Team member management
- Role creation and permission assignment
- Workspace invites with email delivery

## Communications & Outreach
- Invite emails for team members and judges
- Scheduled social posts and social accounts

## Security & Auth
- Supabase authentication (email, OAuth)
- Session handling and protected views
- Public program pages with controlled access
