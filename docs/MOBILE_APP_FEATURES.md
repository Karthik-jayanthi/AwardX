# Mobile App Feature Set

## Core Experience
- Secure sign-in and session management
- Workspace switcher (program-based)
- Role-based access to features

## Submissions & Reviews
- Submission list with status filters
- Submission detail view with files
- Judge assignments visibility
- Quick status updates (if permitted)

## Judging
- Judge task queue (assigned submissions)
- Scoring and rubric entry
- Save progress and submit evaluations
- Offline drafts with sync

## Schedule & Timeline
- Round timeline and key dates
- Upcoming round reminders
- Calendar integration

## Notifications
- Judge invite and assignment notifications
- Deadline and round start alerts
- Status change updates

## Team & Collaboration
- Team directory with roles
- Invite acceptance flow
- In-app messaging (optional)

## Analytics (Lite)
- Snapshot stats (submissions, pending review)
- Progress indicators

## Public Program (Optional)
- Public program page
- Apply/submit from mobile
- Category browsing

## Admin Quick Actions (Optional)
- Approve or shortlist items
- Assign judges to entries
- Trigger invite resend
